# homebrew-tap

Homebrew formulae for [mattwag05](https://github.com/mattwag05) projects.

## Install

```bash
brew tap mattwag05/tap
brew install pippin
# or
brew install rag-quest
```

## Formulae

| Formula | Description |
|---------|-------------|
| [pippin](https://github.com/mattwag05/pippin) | macOS CLI toolkit for Apple app automation |
| [rag-quest](https://github.com/mattwag05/rag-quest) | AI-powered D&D text RPG using LightRAG — runs on small local models (~3B params) |

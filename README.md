# homebrew-tap

Homebrew formulae for [mattwag05](https://github.com/mattwag05) projects.

## Install

```bash
brew tap mattwag05/tap
brew install pippin
```

## Formulae

| Formula | Description |
|---------|-------------|
| [pippin](https://github.com/mattwag05/pippin) | macOS CLI toolkit for Apple app automation |
